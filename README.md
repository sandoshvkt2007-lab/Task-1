
Python To-Do List Application

Description:
This is a simple console-based To-Do List application written in Python.
It allows users to add tasks, view tasks, and remove tasks using a menu-driven interface.
The program runs continuously until the user chooses to exit.

Features:
- Add new tasks
- View all tasks
- Remove tasks
- Simple and beginner-friendly program

Technologies Used:
- Python 3
- Console / Terminal

How to Run the Program:
1. Ensure Python is installed on your system.
2. Save the program file as todo.py.
3. Open terminal or command prompt.
4. Run the program using:
   python todo.py

How to Use:
1. Enter 1 to add a task.
2. Enter 2 to view all tasks.
3. Enter 3 to remove a task.
4. Enter 4 to exit the program.

Sample Menu Output:
--- TO DO LIST ---
1. Add task
2. View tasks
3. Remove task
4. Exit

Concepts Used:
- Lists
- While loop
- Conditional statements
- User input
